🔄 Key Data Flows

1
BMI + Diet: User inputs → FastAPI calculates with scikit-learn TDEE model → MongoDB saves profile → Plotly chart returned to Next.js
2
Posture Analysis: Webcam frame → base64 to FastAPI → OpenCV decode → MediaPipe Pose → joint angles → form score → annotated frame back to UI
3
Live Feedback: Each analysed frame pushes score to Firebase → Next.js client subscribes → UI updates in realtime without polling
4
AI Chat: Messages → FastAPI → GPT-4o (OpenAI) with user context → HuggingFace fallback if quota exceeded → chat history in MongoDB
5
Exercise Classification: Pose image → TensorFlow MobileNetV2 CNN → top-1 prediction → exercise type confirmed to user
6
Rep Counting: Landmark sequence → PyTorch LSTM → rep count integer → displayed on overlay in webcam view
7
Analytics Dashboard: PostgreSQL workout logs → FastAPI aggregates → Plotly figures as JSON → rendered via react-plotly.js + D3 heatmap
8
Media Storage: Annotated frames / video clips → uploaded to AWS S3 → presigned URL returned → displayed or downloaded by user
